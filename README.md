# 记忆翻牌小游戏

一款经典的记忆翻牌游戏，使用纯原生 HTML + CSS + JavaScript 开发。

## 游戏玩法

1. 点击卡牌翻开查看图案
2. 每次可以翻开两张卡牌
3. 如果两张卡牌图案相同，则配对成功，卡牌保持翻开状态
4. 如果图案不同，则卡牌自动翻回背面
5. 目标是用最少的步数和时间完成所有配对

## 功能特性

- 🎴 4×4 网格卡牌布局
- 🎯 流畅的 3D 翻转动画
- ⏱️ 游戏计时功能
- 📊 步数统计
- 🎉 胜利提示弹窗
- 🔄 重新开始按钮
- ⚡ 点击防抖机制

## 技术栈

- HTML5
- CSS3 (3D transforms, animations)
- JavaScript ES6+

## 开发说明

### 文件结构

```
memory-game/
├── index.html    # 页面结构
├── style.css     # 样式文件
└── script.js     # 游戏逻辑
```

### 核心功能

1. **卡牌洗牌**：使用 Fisher-Yates 算法随机打乱卡牌顺序
2. **3D 翻转**：使用 CSS3 transform 实现流畅的翻转动画
3. **配对检测**：比较两张翻开卡牌的图案是否相同
4. **计时系统**：首次翻牌开始计时
5. **胜利判定**：所有卡牌配对完成后显示胜利弹窗
6. **点击防抖**：使用 `isProcessing` 状态防止连续点击造成逻辑错乱
